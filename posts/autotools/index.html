三个主流自动化工具

RobotFramework

Github 地址 Python 语言实现，常用来做App 自动化测试、接口自动化测试、浏览器相关的自动化操作。使用专门的 .robot 脚本来编写自动化的操作，操作浏览器需要安装 Selenium 插件包并设置相应浏览器的 WebDriver 。感觉使用起来比较麻烦。有个专门的IDE：RIDE

AutoMagica

Github Page 也是Python 的一个项目，可以完成操作浏览器、excel 、自动发送微信等工作，可以使用其 Recorder 根据截图来生成移动、点击鼠标等等代码，使用比较方便，但是操作间有延迟，感觉性能不太好。另外 Recorder 会将截图上传至公网，故涉密的界面操作不能使用 Recorder 截图来定位。

AirTest

官网 网易爸爸推出的跨平台UI 自动化编辑器，几乎带UI 的任何东西都能操作，暂时没用过，看上去比 robotframework 好用，待定。
